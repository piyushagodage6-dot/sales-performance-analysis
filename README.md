# Sales Performance & Customer Insights Analysis

## Project Overview
This project analyzes retail sales data to understand sales trends, customer behavior,
and profitability across different categories and regions using Python.

## Dataset
Sample Superstore dataset containing order details, sales, profit, customer segments,
and regional information.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Analysis Performed
- Monthly sales trend analysis
- Category-wise and sub-category-wise performance analysis
- Customer segment and region-wise profitability analysis
- Identification of loss-making products and regions

## Business Insights
- Certain categories drive high sales but lower profit margins
- Specific sub-categories consistently result in losses
- Regional performance varies significantly, highlighting growth opportunities

## Conclusion
This project demonstrates practical data analytics skills including data cleaning,
exploratory data analysis, visualization, and insight generation using Python.
